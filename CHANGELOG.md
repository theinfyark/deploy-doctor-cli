## [1.0.1] - 2026-07-16

### Changed

- Version bump for npm registry publish

## [1.0.0] - 2026-07-15

### Added
- Deployment validation CLI (`deploy-doctor`)
- Checks for Docker, Kubernetes, Helm, Terraform, Azure CLI, AWS CLI, Git, Node
- Disk, memory, ports, and networking probes
- JSON + human-readable reports with exit codes
- Programmatic TypeScript API (`runDoctor`)
